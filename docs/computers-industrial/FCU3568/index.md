# Промышленный компьютер FCU3568

>**Промышленный компьютер на основе одноплатника [NAPI2](/docs/napi2/).** \
>**Работает под управлением Linux (Armbian\NapiLinux).**

## Основные характеристики

- RK3568j, 4 ядра, промышленный диапазон
- HDMI
- USB3.0, USB2.0. USB-OTG
- RS485, CAN
- 2x1Гбит Ethernet
- Питание 9-36В
- Модуль связи (опция)
- Крепление на DIN
