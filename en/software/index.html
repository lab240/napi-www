Книга знаний NAPI (NAPI Handbook)

Практические инструкции по работе с устройствами NAPI: подключение, настройка, работа с датчиками и протоколами. Здесь собраны самые актуальные статьи. Полная техническая база - технические заметки, решения задач и короткие приемы - в Ленте знаний.

  • Прошивка в NAND без кабеля - запись образа через USB-флешку
  • Бекап прошивки из NAND в файл - создание резервной копии

  • Прошивка через кабель (Linux) - прошивка с хоста Linux

Modbus / SNMP / GPIO​

Работа с датчиками и протоколами промышленной автоматизации.

  • Опрос датчика по Modbus RTU через RS485 - чтение датчиков через последовательный порт
  • Опрос датчика по Modbus TCP - чтение датчиков по сети
  • Шлюз Modbus RTU - Modbus TCP - превращение RS485-датчиков в сетевые
  • Сбор данных c Modbus-датчиков через Telegraf - автоматический опрос и запись в базу данных
  • Настройка сервиса SNMP на NAPI - мониторинг устройства по SNMP
  • Установка и настройка Grafana - визуализация данных
  • Установка gpiod и работа с GPIO - управление цифровыми портами

Сети и IP​
