# Заявка на тестирование продуктов NAPI

> **Вы можете взять нашу продукцию на тестирование для
адаптации и отладки Ваших проектов.**

- Napi-C, Napi-P, FCC3308 - как правило доступны для длительного тестирования
- NAPI2 - количество экземпляров ограничено, поэтому возможна задержка.
- Компьютеры FCU3308P, FCUCM4, FCU3568 выдаются под пилотный проект, требуется обоснование.
